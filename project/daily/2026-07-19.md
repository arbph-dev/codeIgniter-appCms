

# Modification infrasctructure
Problème : les routes cms ne sont pas accesibles sans login
  
  ex: [https://zealot.fr/cms/article/test-art](https://zealot.fr/cms/article/test-art)

solution enviagé :
- modifier [filters](/refactoring/app/Config/Filters.php)

soultion confirlmée (grok) :
- Dans Filters.php, tu dois exclure les routes publiques du filtre session.

Voici la version corrigée et propre :
```php
            // === CMS Public ===
            'cms/article/*',      // tous les articles
            'cms/category/*',     // toutes les catégories
            'cms/section/*',      // sections
            'cms/part/*',         // parts
```
