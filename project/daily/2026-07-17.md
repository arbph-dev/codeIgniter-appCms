Le LoaderFactory devient responsable du format
LoaderFactory.load(model)

reçoit :

{
    id     : 'forklift',
    format : 'obj',
    path   : '...',
    mtl    : '...'
}

Puis :

ModelDescriptor
      │
      ▼
LoaderFactory
      │
      ├── obj  → OBJLoader
      ├── gltf → GLTFLoader
      ├── glb  → GLTFLoader
      └── 3ds  → TDSLoader

Le Workbench ne connaît pas les loaders.
