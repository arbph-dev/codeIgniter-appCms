# PHP - Références

Documentation des fonctions et techniques PHP utilisées dans le projet.

## Index par catégorie

### [Arrays](./arrays.md)
- `array_column()` — Extraire une colonne d'un tableau multidimensionnel
- `array_keys()` — Récupérer les clés d'un tableau
- `array_map()` — Appliquer une fonction à chaque élément
- `array_values()` — Réindexer les valeurs d'un tableau
- `array_merge()` — Fusionner plusieurs tableaux
- `array_filter()` — Filtrer les éléments d'un tableau

### [Fichiers et Répertoires](./filesystem.md)
- `file_get_contents()` — Lire un fichier
- `file_put_contents()` — Écrire dans un fichier
- `glob()` — Rechercher des chemins selon un masque
- `scandir()` — Lister les fichiers et dossiers
- `RecursiveDirectoryIterator` — Parcourir récursivement un répertoire
- Helpers — Fonctions utilitaires pour gérer les répertoires
