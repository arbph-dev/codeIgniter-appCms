## 2026-07-07-001: Réintégration du CallOut

Le callout est associé aux 
- classes css dans [app/Views/cms/article.php](/refactoring/app/Views/cms/article.php)
- script dans '/assets/js/ihm/callout.js'         
- structure dans [CalloutRender.php](/refactoring/app/Libraries/Components/Renderers/CalloutRender.php)

### Constat
Le Callout ne fonctionne pas.

Le script initCallout ne trouve pas de class cp_callout titre
Ceci du fait du nommage des classes dans la structure via [CalloutRender.php](/refactoring/app/Libraries/Components/Renderers/CalloutRender.php)


### Evolutions

Pour éviter d'interférer avec la version de production et pour évolution vers le bus
rappel [CalloutRender.php](/refactoring/app/Libraries/Components/Renderers/CalloutRender.php)

```php
$titleHtml = $title ? "<strong class=\"cp_callout_title\">{$title}</strong>\n    " : '';

<div id="{$id}" class="cp_callout cp_callout--{$type}">
    {$titleHtml}<div class="cp_callout_content">{$content}</div>
</div>

```



modifier le script en
```js
	import { initCallout} from '/assets/js/ihm/callout.js'
	import { initCallout} from '/assets/js/components/callout.js'
```
modifier la fonction initCallout pour
- gérer le titre avec la classe cp_callout_title 
- gérer le contenu cp_callout_content


déplacé l'import de app/Views/cms/article.php vers `/assets/js/cms/bootstrap.js`
```
	import { initCallout} from '/assets/js/ihm/callout.js'
	import { initCallout} from '/assets/js/components/callout.js'
```

déplacé initCallout dans initCms() fichier `/assets/js/cms/bootstrap.js`
```js
            initCallout()   // callout : toggles accordéon, pas d'EventBus requis
```
